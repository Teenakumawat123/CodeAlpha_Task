# CodeAlpha Calculator

## Description
A modern and responsive calculator built using HTML, CSS, and JavaScript.

## Features
- Basic arithmetic operations (+, −, ×, ÷)
- Percentage calculation
- Calculation history
- Clear history option
- Dark/Light mode
- Glassmorphism UI
- Keyboard support
- Responsive design

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Run
1. Open the project folder.
2. Open `index.html` in your browser or run it using Live Server.
