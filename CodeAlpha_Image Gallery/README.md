# 📸 CodeAlpha Image Gallery

A modern and responsive **Image Gallery Web Application** inspired by the user interface of a smartphone gallery. This project was developed as part of the **CodeAlpha Frontend Development Internship** using **HTML, CSS, and JavaScript**.

---

## ✨ Features

- 📂 Album-based gallery interface
- 🖼️ Multiple album categories
  - All Albums
  - Camera
  - Screenshots
  - WhatsApp
  - Favorites
  - Pictures
  - College
  - Downloads
- 🔍 Lightbox image preview
- ⬅️➡️ Previous & Next image navigation
- 🗑️ Recently Deleted section
- ♻️ Restore deleted images
- ❌ Delete Forever option
- 🌙 Dark / ☀️ Light Mode
- ✨ Glassmorphism UI Design
- 🎨 Smooth hover effects & transitions
- 📱 Fully Responsive Design
- ⚡ Fast and user-friendly interface

## 🚀 How to Run

1. Download or clone this repository.
2. Open the project in Visual Studio Code.
3. Install the **Live Server** extension (if not installed).
4. Right-click on `index.html`.
5. Select **Open with Live Server**.

The gallery will open in your browser.
